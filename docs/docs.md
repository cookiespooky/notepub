## What Notepub does

- Indexes markdown notes (S3 or local), produces `resolve.json`, `search.json`, sitemaps, robots, and optional collections.
- Serves HTML with a theme and on-demand markdown rendering.
- Builds static HTML for GitHub Pages or any static hosting.

If you are new, begin with **Getting Started**.
