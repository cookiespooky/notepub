Notepub может работать полностью без сервера: `notepub build` создаёт HTML в `dist/`, а вы публикуете его как статический сайт.

В этом режиме нет `/v1/search`, но работает `search.json` и SSR‑страница поиска. Сайт остаётся быстрым и простым для хостинга.
