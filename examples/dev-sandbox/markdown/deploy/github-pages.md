GitHub Pages идеально подходит для режима `notepub build`: вы получаете статический `dist/`, который можно раздать как сайт.

Плюсы:

- бесплатно и надёжно
- нет сервера
- простой CI

Минусы:

- нет серверного поиска `/v1/search`
- динамика только через внешние сервисы

Это хороший старт для личных сайтов, документации и SEO‑блогов.
