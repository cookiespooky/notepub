В Notepub нет базы данных, админки и «скрытых» правил. Есть Markdown, правила в rules.yaml и тема с HTML‑шаблонами. Этот минимальный набор даёт предсказуемый результат: любой файл может быть собран в страницу, любой URL объясняется правилами.

CMS часто добавляют удобства ценой зависимости: данные живут в специфическом формате, логика рассредоточена по интерфейсу и миграции становятся отдельным проектом. У Notepub всё явное: структура — в frontmatter, маршруты — в rules.yaml, внешний вид — в теме.

Если вам нужен контроль, повторяемость и возможность сменить хостинг без миграции контента, этот «минимализм» — именно то, что нужно.
