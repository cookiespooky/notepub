Notepub‑темы обычно идут по одному из двух путей:

- **Vercel‑style**: современная типографика, много воздуха, качественные акценты. Хорошо для продуктовых сайтов и блогов.
- **SEO‑minimal**: почти чистый HTML, минимум CSS, максимальная скорость и индексация. Хорошо для контент‑машин.

Оба подхода держатся на одном каркасе: layout + page/guide/essay/hub. Это позволяет легко менять стиль, не переписывая контент.
