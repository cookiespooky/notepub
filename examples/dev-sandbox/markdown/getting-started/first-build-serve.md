Notepub умеет работать в двух режимах.

- `notepub serve` — рендерит страницы на лету и отдаёт HTML по HTTP. Полезно для разработки и серверного режима.
- `notepub build` — строит статический сайт в `dist/`. Удобно для GitHub Pages и любого CDN.

Оба режима используют один и тот же пайплайн и правила. Разница только в том, где живёт HTML: на сервере или на диске.
