`layout.html` — общий каркас страницы. Внутри него вставляется тело страницы: `{{ .Body }}`.

Обычно структура выглядит так:

- `<head>` с метаданными и подключением CSS
- общий `<header>`
- `{{ .Body }}` с конкретным шаблоном (`page.html`, `guide.html`, `essay.html`, `hub.html`)
- `<footer>`

Этот слой позволяет держать общий стиль и логику в одном месте, а контентные шаблоны — в отдельных файлах.
