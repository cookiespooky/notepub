Минимальный SEO‑шаблон для Notepub обычно включает:

- `<title>` из `title`
- `<meta name="description">` из `description`
- canonical URL на основе `permalink`
- Open Graph / Twitter карточки
- корректную семантику (`main`, `article`, `header`)

Такой шаблон не требует JS и уже даёт достойные результаты. Остальные улучшения — это скорость, структура и чистота HTML.
