Notepub выбирает обычные HTML‑шаблоны, потому что это самый переносимый формат. Нет сборки, нет зависимостей, нет сложной цепочки рендера — только контент и HTML.

Это даёт три преимущества:

- скорость: статический HTML загружается быстрее
- надёжность: меньше зависимостей — меньше точек отказа
- переносимость: тему можно открыть и понять без специфических фреймворков

JSX и React хороши для приложений. Notepub — про публикацию, где ценится простота и долговечность.
