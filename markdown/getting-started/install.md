Notepub — это один бинарник, поэтому установка может быть очень простой. Варианты подбираются под вашу инфраструктуру.

1) Локально: используйте бинарник Notepub и конфиг `config.yaml` в корне проекта.
2) Docker: удобно для серверного окружения, где важна предсказуемость зависимостей.
3) Готовый бинарник: минимальный путь для self‑hosted сценариев.

В любом случае вы получаете одинаковый набор команд: `notepub index`, `notepub serve`, `notepub build`, `notepub validate`.
