Если нужен динамический функционал (поиск по каталогу, калькулятор, бронирование), его можно вынести во внешний API.

Notepub остаётся статическим рендером, а JS подключается точечно. Такой подход сохраняет скорость и SEO, при этом даёт интерактив там, где это оправдано.
